# Rick And Morty Vue

---
The vue.js project I made using the [Rick and Morty API](https://rickandmortyapi.com/) service.

## [Live Demo](https://rick-and-morty-vue.vercel.app/)


## Technologies 
* Vue.js
* Axios
* Tailwind Css
* Vue Router
* Vue Lazylod


## Project setup
```
yarn install
yarn tailwind
yarn serve
```



### Compiles and minifies for production
```
yarn build
```


### Lints and fixes files
```
yarn lint
```
